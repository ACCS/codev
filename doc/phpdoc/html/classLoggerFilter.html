LoggerFilter::decide ( LoggerLoggingEvent $  event)

Decide what to do.

If the decision is LoggerFilter::DENY, then the event will be dropped. If the decision is LoggerFilter::NEUTRAL, then the next filter, if any, will be invoked. If the decision is LoggerFilter::ACCEPT then the event will be logged without consulting with other filters in the chain.

Parameters:
LoggerLoggingEvent$eventThe LoggerLoggingEvent to decide upon.
Returns:
integer LoggerFilter::NEUTRAL or LoggerFilter::DENY|LoggerFilter::ACCEPT

Reimplemented in LoggerFilterDenyAll, LoggerFilterLevelMatch, LoggerFilterLevelRange, and LoggerFilterStringMatch.

Member Data Documentation

const LoggerFilter::ACCEPT = 1

The log event must be logged immediately without consulting with the remaining filters, if any, in the chain.

const LoggerFilter::DENY = -1

The log event must be dropped immediately without consulting with the remaining filters, if any, in the chain.

const LoggerFilter::NEUTRAL = 0

This filter is neutral with respect to the log event. The remaining filters, if any, should be consulted for a final decision.
