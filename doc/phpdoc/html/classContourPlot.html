Detailed Description

This class represent a plotting of a contour outline of data given as a X-Y matrice

Construct a contour plotting algorithm. The end result of the algorithm is a sequence of line segments for each isobar given as two vertices.

Parameters:
$aDataMatrixThe Z-data to be used
$aIsobarA mixed variable, if it is an integer then this specified the number of isobars to use. The values of the isobars are automatically detrmined to be equ-spaced between the min/max value of the data. If it is an array then it explicetely gives the isobar values
$aInvertBy default the matrice with row index 0 corresponds to Y-value 0, i.e. in the bottom of the plot. If this argument is true then the row with the highest index in the matrice corresponds to Y-value 0. In affect flipping the matrice around an imaginary horizontal axis.
$aHighContrastUse high contrast colors (blue/red:ish)
$aHighContrastBWUse only black colors for contours
Returns:
an instance of the contour plot algorithm
