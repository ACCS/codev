static Logger::configure ( $  configurationFile = null,
$  configurationClass = null 
) [static]

Configures log4php by defining a configuration file and/or class.

This method needs to be called before the first logging event has occured. If this method is not called before then, the standard configuration takes place (

See also:
LoggerConfiguratorBasic).

If only the configuration file is given, the configurator class will be determined by the config file extension.

If a custom configurator class is provided, the configuration file should either be null or contain the path to file used by the custom configurator. Make sure the configurator class is already loaded, or that it can be included by PHP when necessary.

Parameters:
string$configurationFilepath to the configuration file
string$configurationClassname of the custom configurator class
